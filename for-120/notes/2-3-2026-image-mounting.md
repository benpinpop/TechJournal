# 2/3/2026: Image Mounting

* Disk Images cannot be opened like traditional files
* You need a tool to analyze its contents
* The device needs to be attached via SATA/SCSI
* We need to assign a drive letter in Windows
* Mounting is the process where OS makes files and directories available
* Unmounting means making all the files unavailable
* Mounting is read-only to guarantee integrity
* Tools may allow mounting with writing mode
* Modifications can be saved to the original image or to a cache file
* Why Mounting
  * Preferable using a file system
  * Easier to run a third party application and do more
* Even though an image is a file, we are still able to treat it like a VHD, and make it accessible through our computer's file system.
