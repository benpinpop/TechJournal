# Glossary

**HDD (Hard Disk Drive)** – A traditional storage device that uses spinning magnetic platters and a read/write head to store and retrieve data. Generally offers larger capacities at lower costs but slower speeds than SSDs.

**SSD (Solid State Drive)** – A storage device that uses flash memory chips with no moving parts, providing faster data access, better durability, and lower power consumption than HDDs.

**USB (Universal Serial Bus)** – A standard connection interface for attaching peripherals like keyboards, mice, storage drives, and other devices to computers. Also provides power delivery to connected devices.

**SATA (Serial ATA)** – A connection interface for attaching storage drives to motherboards, replacing the older PATA standard. Uses a thin serial cable and supports hot-swapping of drives.

**PATA (Parallel ATA)** – An older storage interface using wide ribbon cables to connect hard drives and optical drives to motherboards. Largely replaced by SATA.

**IDE (Integrated Drive Electronics)** – Another name for PATA, referring to the same parallel interface technology. The term emphasizes that drive controller electronics are built into the drive itself.

**SCSI (Small Computer System Interface)** – A high-performance interface standard for connecting multiple storage devices and peripherals in a daisy-chain configuration, commonly used in servers and professional workstations.

Bytes - 8 bits
