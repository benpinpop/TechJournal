# Module 3

* Overview - How does Access Control Work?
  * Controlling Access to Computers
  * Access to Data
  * Access to Networks
* What are we securing against
  * Unauthorized persons, modification, and destruction
* Access Control
  * Locking, Logins, Encryption, and Digital Signatures
  * Denial is just as important as granting access
  * Physical is just as important as Logical Access Control
* Permission Levels
  * Data Owner
    * Owns the Data, determines who has access
  * Data Administrator
    * Maintains the storage of data and who has access
  * Data Architect
    * Design methods in which the data is stored and accessed
  * Vendors
    * Sellers of data
* Hashes
  * To ensure data has integrity
* Use multiple layers
  * Physical, Technical, and Administrative
* Administrative Access Control
  * Policies & Procedures
  * Everyone needs to know what they're responsible for
  * Separation of Duty
* Access Rights
  * Read, Write, Execute, Change/Modify, Delete, etc
    * Owner, Group, Everyone
  * Expire dates
* Policy can cover everything
  * Hiring, Disciplinary, Termination, and Account Setup
* Technical Controls
  * Passwords, Encryption
* Physical Controls
  * Network Segregation
  * Perimeter Security
    * Security Guards
    * Badges, IDs
    * Biometrics
* Access Control Strategies
  * DAC - Discretionary Access Control
    * You specifically
  * MAC - Mandatory Access Control
    * "Government said so"
  * RBAC - Role-Based Access Control
    * Scales very well
  * ABAC - Attribute-Based Access Control
    * Who you are, where you are, etc
* Network Access List
  * Network filter, not as sophisticated as a firewall
  * Simple as permitting or denying hosts inside a network
* AAA - Authentication, Authorization, Accounting
  * Authentication: Verifying users by their profile
  * Authorization: Process of enforcing policies and access
  * Accounting: Measures resources users consume
* IDAM - Identity and Access Management
  * Single ID across organizations
    * Examples: AD, SSO, OAuth
  * Reduces overused passwords
  * Managed centrally
