### Vocabulary & Key Terms

#### Digital Forensics

* The process of examining digital evidence to determine information
  * Typically for cyber crimes and incident response
* Digital Forensics requires a large set of skills from operating system knowledge, to coding, malware analysis, network security, cryptography, law, and ethics.

#### Artifacts

* Anything that can be used as evidence in a case. Artifacts themselves are not called evidence, as they have to be collected and verified properly. Judges and Lawyers decide whether artifacts end up becoming evidence.
* **Common Artifacts**
  * Event Logs
    * Windows
      * C:\Windows\System32\winevt\Logs
    * Linux
      * journalctl
  * File System
    * Use AXIOM Examine and Process to view

#### Patent vs. Latent

* Patent evidence is evidence that is immediately visible
* Latent evidence must be extracted or found
