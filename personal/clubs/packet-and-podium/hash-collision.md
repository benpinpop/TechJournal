# Hash Collision

MD5 has lots of collisions - Any certificate that uses MD5 can be modified and used to impersonate any website

Flame espionage malware - Used MD5 to impersonate Windows Certificates

MD5 is fundamentally broken: You can change data in two separate files in a special way and get the same hash. Even if you append data after, the hashes will still match.&#x20;
